# UniVerse-docker

## Requirements

To run this project, make sure you have the following prerequisites installed:

- Docker (version 20.10 or higher)
- Docker Compose (version 1.29 or higher)
- GNU Make (version 4.3 or higher)

## Usage

Start :

'make up'
